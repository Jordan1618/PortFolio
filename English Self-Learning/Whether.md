# **Whether, full lesson**

Whether means if in the sense of an alternative or uncertainty between two options, not if in the conditional sense.

The classic trap is that in French, si covers both real conditions, like si tu viens, on part, and indirect alternatives, like je me demande si tu viens. In English these two uses need different words.

### If versus whether

| Word    | Used for                                              | Example                               |
| ------- | ----------------------------------------------------- | ------------------------------------- |
| if      | a real condition                                      | If it rains, take an umbrella         |
| whether | an alternative or implicit choice between two options | I wonder whether you're coming or not |

A very reliable rule: if you can add ou non at the end of your French sentence and it still makes sense, use whether, not if. Check whether every GPO is running or not works, because there's an implicit binary choice. If it rains, take an umbrella works too, but there's no natural or not there, so if is correct.

Another useful rule: after a preposition, only whether is correct, never if. The debate about whether GPOs should be enforced, never about if.

Stylistic note: whether is more formal and precise, if is more common in speech for this same alternative sense, but in technical documentation whether is the correct and expected form.
