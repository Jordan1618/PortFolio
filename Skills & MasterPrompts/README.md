# Claude Skills & Master Prompts | Open-Source Library

**A collection of Claude Agent Skills and Master Prompts I've built and tested on real workflows — sysadmin, cybersecurity, automation, and creative production.**

---
## Main Topics

This repository centralizes reusable capabilities I've engineered for Claude: structured **Skills** (SKILL.md-based, task-specific instruction sets) that extend what Claude can reliably do, and a smaller set of **Master Prompts** — dense, reusable prompt architectures for recurring high-value tasks.

---
## Goal of this Library

- **Track Progress:** Keep a clear record of how my Skills evolve from one-off prompts to reliable, reusable tools.
- **Quick Reference:** Build a library of ready-to-drop Skills and Master Prompts I (and others) can reuse instead of re-engineering the same prompt logic every time.
- **Deep Understanding:** Document _why_ each Skill is structured the way it is — not just what it does — so it stays maintainable and forkable.
- **Open Source:** Share what works so others building on Claude don't have to start from zero.
