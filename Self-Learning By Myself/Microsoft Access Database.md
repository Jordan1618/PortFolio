## What is it and how it works ?

#### **1) In professional context i was faced with this tool without knowing it. So it's an opportunity to learn**

- Access is a RAD (Rapid Application Development) that carries in the same place a User Interface and a Data Storage Engine.
- Access has the particularity to connect a lot of element, more than Excel

#### **2) The 4 main objects of Access**

- Tables : Where data is stored with a key system (Primary keys (1/table))
- Queries : Where instructions give life to tables
- UI : To help users while limiting risks
- Reports : The part that converts date into inventories, monthly recaps or bills

#### **3) Other Specifications**

- The Engine is ACE and the program tool is VBA
- Access isn't free, can't exceed 2GB, holds more than 15-20 users and not web friendly
- But Access is very accessible and is well design to develop an application
