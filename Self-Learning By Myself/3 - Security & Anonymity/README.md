## Main Topics

This directory centralizes what I learn about detection, hardening, and staying anonymous online: log monitoring, SIEM-style pipelines, and privacy-preserving tools like Tor.

---

## Goal of this Documentation

- **Track Progress:** Keep a clear record of how my security knowledge evolves from basic hardening checklists to real detection logic in my own monitoring stack.
- **Quick Reference:** Build a library of attack patterns, log signatures, and hardening steps I can reuse instead of re-researching them under pressure.
- **Deep Understanding:** Force myself to explain why an attack works, not just how to block it — this is what separates a technician from an expert.
